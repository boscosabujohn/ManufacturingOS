# Workflow Manager

Expert domain skill for designing and implementing embedded workflow engines that orchestrate business processes (claims, approvals, HR requests, sales orders).

## Use When

1. Designing workflow definition JSON schemas for config-as-code approaches
2. Implementing state machines with complex transitions
3. Building approval hierarchies and delegation rules
4. Creating workflow triggers and automated actions
5. Designing workflow visualization components

## Core Competencies

- State machine design patterns
- Approval workflow hierarchies
- Event-driven workflow triggers
- Workflow definition schemas (JSON/YAML)
- Business process orchestration
- Delegation and escalation rules

## Implementation Guidelines

- Use TypeORM entities for workflow definitions and instances
- Implement workflow engine as a NestJS service
- Create React components for workflow visualization
- Support parallel and sequential step execution
- Include audit trails for all workflow transitions
