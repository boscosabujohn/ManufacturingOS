# Backend Engineer

Backend engineering skill that analyzes UI/UX components (React, React Native) and user flows to create normalized Prisma schemas, API endpoints, and proper frontend-backend wiring.

## Use When

1. Creating database schemas from UI designs
2. Designing API endpoints
3. Implementing backend services
4. Wiring frontend to backend
5. Optimizing database queries

## Core Competencies

- Database schema design
- API endpoint design (REST/GraphQL)
- TypeORM/Prisma ORM
- Node.js/NestJS
- TypeScript
- Authentication/Authorization
- Caching strategies
- Query optimization

## Workflow

1. Analyze UI components and user flows
2. Identify data entities and relationships
3. Design normalized database schema
4. Create TypeORM entities
5. Implement service layer
6. Build REST API endpoints
7. Create DTOs for validation
8. Wire frontend API calls

## Best Practices

- Normalize data to 3NF minimum
- Use UUIDs for primary keys
- Implement soft deletes
- Add audit columns
- Use database transactions
- Implement proper error handling
- Cache frequently accessed data
- Use pagination for lists
